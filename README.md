# rif
A tool to uncover and resolve the effects of weak memory on a binary executable.

Background on the tool, why it exists, & the theory behind it, is all available in [thesis.pdf](https://github.com/UQ-PAC/rif/blob/main/thesis.pdf).

Practical information for how the tool is written, future directions, problems, and a file-map are available in [this repository's wiki](https://github.com/UQ-PAC/rif/wiki).
